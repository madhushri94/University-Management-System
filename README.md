## 📖 Project Description
The University Management System is a desktop-based application developed to manage university operations efficiently.

It helps in managing:
- Student records
- Faculty records
- Leave details
- Course details
- Marks management
- Fee Structure
  
## 🛠 Technologies Used
- Java 
- MySQL Workbench
- JDBC
- NetBeans

This project was developed as a Final Year Project for Bachelor of Computer Science (BCS).
